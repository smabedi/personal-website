# Personal Portfolio

The personal portfolio and web presence of Seyed Mahdi Abedi, an undergraduate Computer Engineering student at Sharif University of Technology.
Currently, at smabedi.ir

This repository contains the source code for the website, featuring custom UI elements, interactive graphics, and custom JavaScript effects.

## 🛠️ Tech Stack
* **HTML5** - Structure and semantics
* **CSS3** - Styling, animations, and responsive design
* **Vanilla JavaScript** - DOM manipulation and custom interactive cursors

## 🚀 Local Development
This project is built to run cleanly without a heavy local build step.

1. Clone the repository:
   ```bash
   git clone https://github.com/smabedi/personal-website.git
   ```
2. Open the project in IntelliJ IDEA (or your preferred IDE).
3. Use the built-in IDE web server to preview `index.html`, or utilize the Live Edit feature for instant updates.

## ⚙️ Deployment
This project utilizes a Continuous Deployment (CD) pipeline via GitHub Actions. Any changes pushed to the `main` branch are automatically synced to the production server via FTP.
